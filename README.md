# 📧 Spam Email Classifier

A Machine Learning project that classifies SMS messages as **Spam** or **Ham (Not Spam)** using Python and TF-IDF Vectorization.

---

## 🚀 Project Overview

This project uses Natural Language Processing (NLP) and Machine Learning to detect whether an SMS message is spam or not.

The model preprocesses text messages, converts them into numerical features using TF-IDF, trains a classifier, and predicts whether a new message is spam.

---

## ✨ Features

- SMS Spam Detection
- Text Preprocessing
- Lowercase Conversion
- Punctuation Removal
- TF-IDF Vectorization
- Machine Learning Classification
- User Input Prediction
- High Accuracy

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

---

## 📊 Dataset

SMS Spam Collection Dataset

Total Messages: **5572**

- Ham: 4825
- Spam: 747

## 📈 Model Performance

Model Accuracy:

**95.70%**
